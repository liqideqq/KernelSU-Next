KernelSU Next

A kernel-based root solution for Android

Supports kernel 4.4 - 6.6 (Non-GKI & GKI)

Installation

Run the following commands below for different installation options, should be ran inside the root of your kernel directory.

curl -LSs "https://raw.githubusercontent.com/rifsxd/KernelSU-Next/next/kernel/setup.sh" | bash -
curl -LSs "https://raw.githubusercontent.com/rifsxd/KernelSU-Next/next/kernel/setup.sh" | bash -s next
curl -LSs "https://raw.githubusercontent.com/rifsxd/KernelSU-Next/next/kernel/setup.sh" | bash -s v1.0.2-R14
