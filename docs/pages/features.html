Features

  • ✔️ Non-GKI kernel support from 4.x - 5.4 LTS (3.x is experimental).
  • ✔️ GKI kernels support LKM/KMI methods for easy installation.
  • ✔️ Magic Mount ensures fast, light, and convenient module mounting.
  • ✔️ New manager features and UI overhaul.
  • ✔️ Restoration of modules post-uninstallation.
  • ✔️ Manager app auto-updates with new releases.
  • ✔️ Frequent updates and upcoming features.
